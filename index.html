인마고 리눅스 연습용

It works!

리눅스 웹 서버 설정 연습용으로 만들어진 페이지입니다. Apache2가 정상적으로 작동하는지 확인하기 위해 사용됩니다.

만약 이 페이지를 볼 수 있다면, Apache HTTP 서버가 해당 사이트에서 제대로 작동하고 있음을 의미합니다. 웹 서버를 계속 사용하기 전에 이 파일(/var/www/html/index.html)을 교체하십시오.

Configuration Overview

Ubuntu의 Apache2 기본 설정은 원본 설정과 약간 다릅니다. Ubuntu 도구와의 상호 작용을 최적화하기 위해 여러 파일로 분할되어 있습니다. 설정 시스템은 /usr/share/doc/apache2/README.Debian.gz에서 완전히 문서화되어 있습니다. 자세한 내용은 이 문서를 참조하십시오. 웹 서버 자체에 대한 문서는 apache2-doc 패키지가 이 서버에 설치되어 있으면 manual을 참조하여 확인할 수 있습니다.
